# my-web-project

一言日記アプリです。

## 概要

1 日ごとに一言の日記を保存できる、シンプルな Web アプリです。
Android 端末での利用を想定し、以前使っていた日記アプリの UI を再現しました。
Android Studio を使用し、Web 技術で作成したアプリを Android 端末向けにパッケージ化しています。

## 作成目的

- HTML/CSS/JavaScript の基礎の復習。
- 新しいスマートフォンでも同じ日記アプリを使えるようにするために作成しました。

## 使用技術

- HTML
- CSS
- JavaScript
- ローカルストレージ

## 機能

- 日記の保存
- 横スワイプによる月の切り替え

## 工夫した点

- 学習補助として生成 AI の提案コードを参考にしつつ、理解しながら実装しました。
- 復習と学習を兼ねてコードのコメントを多めに記述するように心がけました。
